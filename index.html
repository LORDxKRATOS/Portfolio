    Face Detection in Real-Time

    This project implements a real-time face detection system utilizing Python and OpenCV, designed to identify and track faces with high accuracy in live video streams. The primary objective is to provide an efficient, responsive application that processes video data from a webcam or camera feed, detecting human faces in real time. The system leverages the Haar Cascade Classifier, a machine learning-based approach where a cascade function is trained from a large number of positive and negative images. Once trained, this cascade can efficiently detect objects—in this case, human faces—in video frames.

    Netflix Clone

    The Netflix Clone project is a front-end recreation of Netflix, built entirely with HTML and CSS. It replicates the iconic design of the streaming platform, featuring a responsive layout with multiple sections like "Trending Now" and "Top Picks," each displaying movie posters in a grid or row format. The project includes a sticky navigation bar, a large hero banner, and hover effects on movie posters and buttons, all styled to closely resemble Netflix's dark theme and red accents. Although static, the clone demonstrates the power of CSS in creating visually engaging, responsive web pages without JavaScript or backend integration.
